case class CreatePassiveSellOfferOperation(selling: Amount, buying: Asset, price: Price, sourceAccount: Option[PublicKeyOps] = None) extends Operation with Product with Serializable

Creates an offer that won’t consume a counter offer that exactly matches this offer.

selling

the total amount of tokens being offered

buying

the asset being sought

price
